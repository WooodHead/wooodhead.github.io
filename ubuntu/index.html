# ubuntu18禁用时间锁屏界面

  1. Disable Screen Lock
    Gnome settings → Privacy → Screen Lock → Off

首先在系统设置里禁用锁屏

  1. Install this extension
    Go to "Ubuntu Software" application.
    Search for "disable screen shield".
    Install it.

下载安装 disable screen shield

  1. Reload GnomeShell
    Press ALT + F2 then enter r and press Enter.

按 ALT+F2 然后输入 r，回车。这样会重新载入桌面

  1. Activate this extension
    Tweak Tool → Extension → Disable Screen Lock → On

使用Tweak Tool 管理这个插件的开关

安装 tweak-tool:

sudo add-apt-repository universe

sudo apt install gnome-tweak-tool
